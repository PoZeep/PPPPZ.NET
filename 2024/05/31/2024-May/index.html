  • 早上过去签个道就去自习室学一个小时再回去整实验报告
  • 下午高精度三小时学习挑战

要说具体目标其实每周五不好说，也不方便把什么复习任务就只放在周五这天，所以不如高精度按考研时间就学三小时，这样无论学什么自由的，这项计划我称为周五可塑性

5.18 周总结日！

遇见讲的课…哎，整个OS都是他给我们上，在之前的随笔中我已经多次提到这个老师的讲课水平问题了，也许也是我有些偏见，但我感觉真的讲很一般，这是个文字型老师，说实话ppt文档什么的做的都挺好，但上课差不多80%念ppt，速度又慢，还经常讲错，根本不适合讲课

所以不会跟他直播课了，所以这里的操作系统学习方式我将会改成 先过一遍ppt再1.5倍速过直播课

今日其实心静不下来，像昨天一样，其实出现也不是第一次了，这个时候两大原则就是 远离电子设备 + 冥想或运动，于是今天回来又做了一个25分钟冥想状态好多了，说实话久违的感受到了那份静

今天也是国赛开赛的第一天，没想到今年已经变成pycc分站赛了，虽然我也不打，但和群友在那瞎扯.jpg，其实很失望，我第一次对CTF感到有点绝望就是两年前打的这场国赛，当时打完心力交瘁，RE是阅读题抽象树AST，其他基本做不出来吧，一道签到题给干到几分…也是第一次对CTF感受失望，而如今彻底变成和PYCC一个性质的了，不敢想象之后的国赛会变成什么样子，反正我是不打国赛哩，打比赛也要有目的性，能从这些题目中学到什么，复现什么题目有成长，不过这些也是后话了

5.19 我能在河边学上一整天的计组🎣

做了一题23年数二的积分几何应用的题，为啥这么简单？？甚至没什么计算量，我以为真题最低水准都没有一道送分题，看来卷子上还是有一些基础分的

今天计组K.O!课上的所有内容都过了，剩下一点知识点就是强化的时候过了

每学完一本都能攒出这么多，这个月再把这些知识点做成脑图，完成计组大一统！学习知识的过程就是打碎知识点，再重组成自己的脑图，再刷题与不断复习达到一定高度！

今天听了遇见的CU与ALU，讲的真的是依托答辩，而整个OS都是他上，昨晚听他直播课已经感觉不妙了，说实话准备OS跟王道了，上课讲不明白，没激情，没水平，念PPT流水线，我真搞不懂01机构这么几年他是怎么不被人吐槽的

今天说实话状态很好？精力很好，吃那个鼻炎和结膜炎的药好像会导致有点嗜睡，反而让我周五周六的睡眠都很充足，于是今天状态满满，狠狠的将精力喷射到了计组上，主要是感觉自己有点落进度了，想把计组学完再开OS，现在好了，明天就是线代+OS了，爽的，今天也是学的最晚的一天，回到家都11点了，也差点被关学校里.jpg

之后数据结构我就称DS，计组就是CC，操作系统就是OS，计网就是CN，DS的脑图等我CC做完就开

5.20 我能在河边听上一整天的张宇线代🎣

真的是相声课，说实话每一小节都有笑点，我真的绷不住了，不过真的很佩服张宇老师啊，教授不光是数学，也是教做人，相声课所折射出来的研究生思想也是非常值得深思的

线代第一讲内容也不难，之前倒也都接触过，争取这五天把一二讲解决了

今天才是开了OS的头，说实话我到现在还不知道OS在学啥，不过可能是因为和我逆向的内容很多想象的关系了，实际上概述这一大章大部分都是概念，而这些概念我基本都知道.jpg，所以我就感觉只是在讲一些前述知识，然而这好像就是本章的内容了，感觉每个学二进制的都是先天408圣体了xs

今天感觉看了讲义发现比直播课思绪清楚的多，果然遇见的课不像M佬的课只需把课上内容反复规划，遇见的精华在讲义上，M佬的精华在课与题，目前还是跟着遇见走吧，讲义看的还是挺舒服的，后续再加个1.5倍过直播课，对遇见印象不好的主要原因还是他在DS讲的kmp和CC讲的运算部分太乱了，然而他是OS的主讲，也许在这个部分才是他真的有东西的领域吧

今天说实话没睡够，不过学习不能总是追求一个好的状态才学，这也是我的一个业力，总是想追求一个好的状态才开始学习，而状态不好的时候就想逃避，这个问题也伴随了很久了，也许就是从小时候就养出的习惯，但每一次的备考都是一次学习行为矫正，现在每天的学习时间我感觉都不够用，尤其这周更加明显，我会有意识的感觉到time up!于是不肯就结束这一天的学习，于是通过各种方式在短时间内恢复自己的状态开始解决这个问题，我觉得也是这次备考发现自己一个非常大的一个问题的矫正

刚好在520这天看到线代第一讲，真的好巧

5.26 Day One

这篇我很喜欢，他们都说挺难，不过我感觉还好，最后两题差不多都是送分题感觉，前三题与我比较有共识，所以感觉做的还是比较顺的

今天是发现了，不同老师的学习方式不同，不得不说遇见做的PPT与讲义真的很不错，然后就想起了我讲课的方式好像也是这种模式，先写好文档型的，然后再根据写的内容围绕开展讲，害，于是我可以总结一下01三个老师的学习方式

罗素：数据结构

  1. 讲义大致翻一下预习
  2. 跟直播课
  3. 直播课回放复盘 + 讲义与PPT + 王道精读
  4. 例题与习题 复盘

三月份学的了，我现在只记得当初是这个过程了

M佬：计组与计网

  1. 直接跟直播课
  2. 直播课复盘 + 讲义 + PPT
  3. 王道精读
  4. 例题与习题 复盘

M佬直播课值得反复听，每次听都有不同的感悟，说实话真的挺佩服这个老师的直播课，不过讲义做的就差强人意了，排版比较乱，字也很小

遇见：操作系统

  1. 精读讲义
  2. 直播课回放 + 讲义笔记与PPT
  3. 王道精读
  4. 习题 复盘

说实话遇见放的题有点少，文字表达型老师，直播课如果没有高精度容易睡着，他的课更像是配合讲义和PPT，而不是讲义配合课，因为我在第一遍读讲义基本就能理解大部分知识点了，就是需要回放直播课来做笔记与加深影响，不过OS学到现在也算是刚入门吧，不过感觉都是概念理解性的问题

每个老师讲课方式与习惯都不同…导致我本书切换来适应老师会发现之前的学习方法有点不通用，不过还在现在就知道如何跟他们强化轮的课了，不过计网也是M佬讲，可以期待一手

今日拖着昨天被1000米创烂的身体去学习了，说实话身体真的好酸，而且周常周六到周日晚上睡不着，不过今天也算是成功学习了一天，中午杭州闷到爆炸，是真的爆炸，也很潮湿，下午终于下了一场暴雨，晚上雨后的空气真好闻，学校高楼也不多，在操场上看雨后的云也很舒适
