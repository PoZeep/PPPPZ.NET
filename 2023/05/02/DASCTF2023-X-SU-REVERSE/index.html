DASCTF2023 X SU REVERSE

easyRE

0x00 Routine Check Shell

非常明显的python打包成的exe，拿github上的脚本嗦了

https://github.com/extremecoders-re/pyinstxtractor

0x01 Decompilation

随后拿到pyc文件发现uncompyl6或是pycdc无法反编译，而现在出现了好用的gpt，直接用pycdas读出字节码丢给gpt翻译即可

丢给gpt即可翻译成

0x02 Get Flag

所以程序只是将我没的明文异或了个随机数，而种子是已知的，也就是这个随机数我们可以知道的，直接异或回去即可

gotots

0x00 Routine Check Shell

Go逆向，无壳

0x01 Analysis Main

我7.7不知道为什么抽风识别不出来，所以上了官网8.2的Free版，于是直接拉入ida，测试程序发现会跑出Incorrect字符串，所以直接shift+ F12搜索定位到字符串再交叉引用主函数，于是可以直接在主函数动调找逻辑

调试后不难发现我们输入只是被异或了个值，那么拿到密文和key就可以逆向出我们的flag

0x02 Get Flag

那么通过调试拿到直接异或即可

multi-universe

0x00 Routine Check Shell

无壳

0x01 Analysis Main

这题如同上题，IDA编译不怎么好看，可以直接下读写断点找到加密函数，或者直接调也可以发现每次经过 main__Cfunc_sub_badcoffee 就会异常然后我们的输入就变了，不难想到有个异常处理函数就是加密我们输入的地方了，于是我在输入上下了个读写断点定位到加密函数

加密完这一段后后还有个循环左移3位再异或0xC，写出加密就是

0x02 Get Flag

那么加密过程知道了，密文直接后面拿，直接解密即可
